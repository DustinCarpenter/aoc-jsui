Day

Inputs — Each day uses a single input shared by both parts. You can paste your personal puzzle input directly into the input box, or load a local example file if one exists. If Auto-load example input is enabled in Settings and a file is found at {inputsPath}/{year}/{day}-input.txt, it will be loaded automatically when the day opens. Your input is never saved by the app and only exists in your browser session.

Solutions — Solutions are plain JavaScript files. If Auto-load solutions is enabled in Settings and a matching file exists at {solutionsPath}/{year}/{day}-1.js or {solutionsPath}/{year}/{day}-2.js, it will be loaded into the editor automatically. If no day-specific file is found, a default starter template is used instead. Your code receives the full input text as input, and whatever value you return is shown in the output box. Use the save button to download your solution file when ready.

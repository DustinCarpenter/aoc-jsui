Day

Inputs - For each day an example and personal input can be used. If you save your input (full or example) to {inputsPath}/{year}/{day}-input.txt, it can be loaded into the input box when the day opens (if auto-load is enabled).

Solutions - Solutions are plain JavaScript files stored at {solutionsPath}/{year}/{day}-1.js and {solutionsPath}/{year}/{day}-2.js. The editor does not save automatically, use the save button to download your file, then place it in the correct folder. If auto-load is enabled and matching files exist, they will be loaded into the Part 1 and Part 2 editors when the day opens. Your code receives the full input text as input; whatever you return appears in the output box.
